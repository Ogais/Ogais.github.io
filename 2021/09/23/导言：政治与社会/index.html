一、马克思主义哲学与政治哲学

-

什么是政治？

不是日常的 “政治化” 的意思。

亚里士多德：政治是 共同体的普遍事务。
词源：政治 与 城邦 是一样的。
“人是 城邦/政治 的动物 ”

施密特：批评-政治 和 国家 的循环论证：国家的大部分事务都是 “行政” 事务，那么行政事务=政治事务？

如何理解政治共同体追求的 普遍性 ？

共和主义的政治概念

共同体的普遍性目标是在政治活动中产生的，而不是先于政治而存在。

罗尔斯：
完备的程序正义（程序本身决定了什么是正义，正义不外在于程序存在，我们不知道什么是正义，但是遵循程序就能达到正义）；
第二种-我们知道且能通过程序达成；
不完善的程序正义（既不知道，也不一定能达成）

共和主义像第一种程序正义。

问题在于，我们如何保证政治活动能最终产生普遍性目标？从主观和客观进行设定

政治的优先性

共同体：共同体比个人更代表善，“整体先于部分”（亚里士多德）

卢梭：政治自主-不仅仅是消极自由（不受干预的自由），而是服从于自己所立之法（不是支配关系，而是 服从 于自身的意志/普遍意志）
公意（普遍意志）与众意，自由的反面是服从而非限制，x众意-服从于他人的结果必然是失去自由
问题：预先设定了一个超脱的公意（普遍意志）？

黑格尔：政治国家（伦理国家）所能实现的价值-超越个人。
在政治国家中，普遍性与特殊性统一起来，客观精神达到自在自为的普遍性，自由获得真正的现实性。 回应卢梭-什么是普遍意志：普遍性并不与特殊性相对立
个体的满足，国家更高的伦理价值：只有在国家中才能实现真正的自由

卢梭与康德的比较：http://www.philosophy.org.cn/fzxk/zzzx/201803/t20180329_3891857.shtml

-

自由主义的政治概念与个人权利的优先性

从个体的自然权利出发，而不是共同体的政治优先性。建立国家、公共参与的目的是保护个人的自然权利。

政治成为派生的、工具性的活动。

康德

政治是 “践行的权利学说”，指向外在的自由（人的自由可以依靠一条普遍的观念而共存）
预先设定了政治的目标：个人自然权利的实现。

“在经验层面实现出自实践理性的先天立法的普遍权利原则的实践活动”

人的权利与公民的权利，哪个优先？（哈贝马斯《？》：政治自主与个体自主的合流）

现实主义的政治概念

马基雅维利：去道德化的政治；政治的核心是权力的夺取和自我持存
历史背景：佛罗伦萨四分五裂，没有建立的国家秩序。

施密特：政治的核心在于区分敌我
反对亚里士多德 政治=国家 的观点：政治概念先于国家概念，国家概念依赖于政治概念
——共同体能作为国家=可以自主区分朋友和敌人，并在必要的时刻消灭敌人

-

政治是共同体的 普遍 事务:
| | 共和主义 | 自由主义 |
| —— | —————— | ————– |
| 出发点 | 共同体的政治优先性 | 个体的自然权利 |
| 政治的地位 | 优先性，超越个人的价值 | 派生性、工具性的活动 |
| 普遍性目标 | 不知道，在政治活动中产生（程序正义） | 先于政治活动存在（人的自然权利的实现） |

马克思

-

（两个命题的对比：政治 的区别？）

马克思早期的政治概念

“在真正的民主制中，政治国家 就消失了”（第一个命题）
黑格尔意义上的政治概念：与人的现实物质生活（利己主义）相分离的抽象的普遍性观念

马克思后期的政治概念

“当阶级差别在发展进程中已经消灭而全部生产集中在联合起来的个人的手里的时候，公共权力就失去政治 性质。”
政治不再是抽象领域中的普遍性，而是由虚假的普遍性所掩盖的阶级斗争和阶级统治关系
——政治：阶级统治和阶级斗争
-

施密特和马克思

共同点：都是现实中的政治活动的逻辑（施密特-区分敌我；马克思-阶级斗争），强调敌对。
政治是为应对群体之间不可调和的矛盾而产生的。

差异：
1、马克思主要关注阶级矛盾；施密特则认为，一切有组织的单元的矛盾，不管是经济的、宗教的，只要激化都可能转化为政治矛盾。
2、施密特认为，敌人的可能性是始终存在的，因此和平主义的方案是去政治化 的（愚蠢）方案，最终导致无意义。（甚至有霍布斯-走出自然状态，回到常规秩序）马克思的共产主义学说可以说是最彻底的去政治化学说，认为共产主义社会中一切阶级对立都会最终消灭，最后政治随着国家而消失。
3、施密特批判的矛头是政治思想的工具化、中立化（规范决定一切，一切都有规范）（“公法到此力所不能”），回避决断，依靠议会，拒绝对人生在世的根本问题作出回答。自由主义认为一切有法可依，拒绝超程序的主体。施密特认为，价值中立化最彻底的就是经济决定，而这就是马克思。施密特的主要对手是自由主义，但马克思同样是他敌对的对象。

二、公民社会与市民社会

-

civil society：公民社会 还是 市民社会？

自然权利学说中：公民社会
公民社会 = 政治共同体本身，承认私人生活但不是关注重点

共和主义传统：不关注市民生活，消极的、被公共生活排斥的东西
古典自由主义：警惕国家权力侵入个人生活，对市民生活缺乏关注

黑格尔：市民社会作为需求的体系
《法哲学原理》：
建立在劳动基础上，交换形成的依赖体系。
受自然必然性所支配。仍然是每个人对每个人的战争。
不等同于政治共同体。

市民社会 与 政治国家 的分离

黑格尔：现实的（国家与社会在结构、功能和概念上的分化） 和 概念上 的两种分离
（马克思青年时期没有搞明白两种分离。

现实的分离：人的经济生活超出了家庭和局部的关系，成为越来越普遍的人际生活形式，无法纳入政治生活，但越来越显著地塑造着现实生活的面貌。
理论上的反映-政治经济学超出亚里士多德家政学的范畴，也不在传统政治学之中。在概念上的分离。

规范性的分离：政治国家应该作为普遍性领域超越市民社会的特殊利益纷争之上，仅仅以具有真正的普遍性的伦理价值作为其目的。

《在传统与革命之间》朱学平译

三、历史与历史哲学

-

康德

历史哲学：反思判断力 在历史领域运用的结果
判断力：将普遍性与特殊性连接的能力
（1）规定的判断力：用自然法则决定经验（将普遍规则适用于特殊情形）
（2）反思的判断力：从特殊事物中寻找适用于它们的普遍规则（寻找普遍目的）

历史的合规律性问题：历史经验背后是否有某种规律？
历史的合目的性问题：历史的合规律性是否体现了某种规范性、合目的性？

康德认为，即使历史中有种种的恶，历史仍然是人类朝向个体权利的实现的进程。
这样的实现需要秩序，即永久和平的全球法治秩序。
这样的朝向、这样的进程需要动力，动力不是源于人类道德的进步，而是人的自我启蒙——人在社会生活中对自我利益的追求。

为什么人先天立法的目标会和历史的朝向逐渐合一？
（莱布尼茨：先天和谐）
受亚当斯密的启发：看不见的手（康德的 “天意”）；
曼德维尔：个体的恶与公共的善
康德认为，人追求个体的利益，因而唤醒自己的潜力，促进其禀赋得以发展，并推动永久和平的全球秩序的实现。
-

黑格尔

历史哲学不是原始的历史，也不是反省的历史，它是对历史的思想考察，是在思想中把握历史进程之中所包含的合理性。
哲学视野中的世界历史是合理化的进程，但它所研究的又不是应然的历史，而是世界历史所已经现实地揭示出来的合理性。

“合乎理性地把握历史” 的含义：
1）历史在理性中：历史的发展中有一种理性的逻辑结构。这种逻辑结构是在人的理性的认识和实践活动中展开的，和上帝或者观察者无关；
2）理性在历史中：理性自身的展开只能是历史性的，人对世界的合乎理性的认识依赖于经验材料和由此揭示的可能性，任何人都不能外在于此对世界做进一步的合乎理性的认识。

历史是，设定出的理性原则不断实现的过程。同时，历史的进步不是线性的，历史的起点不能揭示历史的终点，人对自由的认识及对自由的实践的认识都是受局限的、片面的。片面就有对立，历史的发展就是去解释并消灭对立。

黑格尔认为，我们不能脱离现实去揭示什么是合理的。现实-合理-抱一，逻辑和历史的统一。不是把逻辑强加于历史。

理性的展开只能是历史性的。

康德和黑格尔的不同：（1）康德认为线性，黑格尔认为非线性、辩证。（2）康德的历史哲学朝向未来，黑格尔拒绝对未来做出预测，认为思想能把握的只是已经发生的事情。
